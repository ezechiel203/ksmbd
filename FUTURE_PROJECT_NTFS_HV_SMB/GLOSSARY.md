# Glossary

## Terms
### Program A
Minimum credible Hyper-V over SMB on Linux-backed storage.

### Program B
Maximum Windows Server SMB + NTFS + Hyper-V parity program.

### RSVD
Remote Shared Virtual Disk behavior used for shared-disk Hyper-V scenarios.

### VHDX
Hyper-V virtual-disk file format.

### NTFS Parity
Behavioral compatibility for Windows-visible filesystem semantics beyond basic file I/O.

### Claim Boundary
The exact scope of what the product is allowed to say it supports.

### Interoperability Lab
The Windows/Hyper-V validation environment that gates support claims.
