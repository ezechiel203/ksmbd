# B0-T1 Program B Approval Memo

Status: `planning`

## Purpose
Record the only valid authorization to move Program B from planning to funded
execution.

## Required Outputs
1. Explicit yes/no approval
2. Business justification
3. Staffing commitment
4. Lab funding commitment

## Approval Rule
Program B remains unapproved unless this memo names an executive owner, funding
source, scope boundary, and release-governance owner.

## Exit Criteria
1. Approval or rejection is written and durable.
2. Any approval cites the exact branches in scope: NTFS, RSVD/VHDX, failover,
   or a constrained subset.
